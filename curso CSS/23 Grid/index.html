grid

grid nos permite organizar toda nuestra pagina web en base a columnas. Al igual que flex vamos a necesitar una etiqueta que haga de contenedor como un div y los hijos del contenedor seran los que se vean afectados.

formas de escribirlo

en css vamos a utilizar los siguientes elementos para poder crear filas y colimnas que son :

  • columnas :

      • En este caso por cada 12px que tenemos sera una columna que se crea y ademas del espacio que tendra esa columna en este caso 12px.

    • grid-template-columns: 12px 12px 12px

      • Tambien lo podemosponer de forma mas abreiada posible

    • grid-template-columns: repeat(3,12px)

      • Otra forma posible es la de poner auto para que sea el html que ajuste el espacio por cada auto sera una columna y se puede poner tambien de las dos formas e incluso combinadas

    • grid-template-columns: auto auto auto
    • grid-template-columns: repeat(3, auto)
    • grid-template-columns: 12px auto 12 px

      • tambien se puede poner con porcentajes %

    • grid-template-columns: repeat(3, 5%)
    • grid-template-columns: 12px auto 5%

  • filas :

      • En este caso por cada 12px que tenemos sera una fila que se crea y ademas del espacio que tendra esa fila en este caso 12px.

    • grid-template-rows: 12px 12px 12px

      • Tambien lo podemosponer de forma mas abreiada posible

    • grid-template-rows: repeat(3,12px)

      • Otra forma posible es la de poner auto para que sea el html que ajuste el espacio por cada auto sera una columna y se puede poner tambien de las dos formas e incluso combinadas

    • grid-template-rows: auto auto auto
    • grid-template-rows: repeat(3, auto)
    • grid-template-rows: 12px auto 12px

      • tambien se puede poner con porcentajes %

    • grid-template-columns: repeat(3, 5%)
    • grid-template-columns: 12px auto 5%
